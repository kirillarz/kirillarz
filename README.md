## 🛠️ Стек технологий

Проект создаётся на современном React-стеке с упором на выразительный интерфейс, производительность и возможность спокойно развивать сайт дальше.

### Основной стек

* **Next.js** — фреймворк для приложения, маршрутизация, SEO и серверные возможности
* **React** — компонентный подход к разработке интерфейса
* **TypeScript** — типизация и более надёжный код
* **Tailwind CSS** — стилизация и адаптивная вёрстка
* **shadcn/ui** — готовые доступные UI-компоненты: формы, диалоги, tooltip, drawer и другие элементы интерфейса

### Анимации и визуальная часть

* **Motion** — анимации при скролле, hover-эффекты, переходы между страницами и интерактивные элементы
* **Three.js** — 3D-графика для интерактивного hero-блока
* **React Three Fiber** — интеграция Three.js со стеком React
* **@react-three/drei** — набор готовых помощников и компонентов для работы с 3D-сценами

### Контент

* **MDX** — хранение описаний проектов, статей и будущих материалов блога рядом с кодом проекта

### Качество кода и тестирование

* **ESLint** — статический анализ кода
* **Biome** — единый стиль форматирования
* **Playwright** — end-to-end тестирование ключевых пользовательских сценариев

### Развёртывание

* **Docker Compose** — контейнеризация окружения для локальной разработки и продакшена
* **VPS** — самостоятельное развёртывание проекта на сервере
* **GitHub Actions** — автоматическая сборка и деплой после push в основную ветку

### Планируемые расширения

* **Payload CMS + PostgreSQL** — self-hosted админ-панель для управления проектами, статьями и медиафайлами
* Форма обратной связи с серверной обработкой и уведомлениями в Telegram
* Аналитика посещений
* Раздел с блогом, кейсами и подробными описаниями проектов
